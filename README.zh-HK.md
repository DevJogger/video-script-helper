# 視頻稿件處理工具

一個用於處理和格式化視頻稿件的網頁應用程式，支援粵語、國語和字幕生成。

## 功能特點

- **多種輸出模式**：
  - **粵語（粵）**：將文本轉換為粵語錄音稿，會替換部分原稿字詞為粵語慣用字詞，如“公寓”會被替換成“柏文”
  - **國語（國）**：將文本轉換為國語錄音稿，會替換部分原稿字詞為國語慣用字詞，如“廿”會被替換成“二十”
  - **字幕（字幕）**：生成可下載的字幕格式文本
- **發音提示**：開啟/關閉容易讀錯的粵語發音提示（測試版）
- **打印支援**：打印國語稿件時可選擇添加拼音注音
- **下載功能**：以 UTF-16LE 格式匯出字幕檔案

## 使用方法

### 開始使用

1. 在網頁瀏覽器中開啟應用程式
2. 您會看到兩個面板：左側的**輸入區**和右側的**輸出區**

### 輸入稿件

1. 在左側的**輸入區**編輯器中輸入或貼上原始稿件
2. 所有在右側輸出區的修改都不會改變左側輸入區的原始稿件

### 轉換稿件

1. 點擊輸出面板中的按鈕選擇輸出模式：
   - **粵** 粵語模式
   - **國** 國語模式
   - **字幕** 字幕模式

2. 轉換後的文本會自動顯示在**輸出區**編輯器中
3. 粵語/國語模式會替換預設的字詞，被替換的字詞會在右側的輸出區用紅色字標記，便於檢查是否會出現語義偏差
4. 如果在替換後出現語義偏差，比如“目的地”在轉為粵語錄音稿時被替換成“目嘅地”，可以在左側的輸入區對字詞添加下劃線，則會避免被錯誤替換，此外也可以直接在右側編輯修改
5. 字幕模式下，右側輸出區會將不包含中文的段落自動刪去，並在逗號句號的位置刪去標點，並自動換行，頓號會被替換為空格
6. 字幕模式下，同樣可以直接對右側輸出區直接進行編輯，不會改變原始稿件，輸出區的背景在第16個字之後的位置變為灰色，便於作為字幕編輯的換行依據

### 調整設置

點擊輸出面板中的**設置**圖示（⚙️）來存取：

- **粵語發音提示（測試版）**：切換不同發音的提示，默認全關
- **打印設置**：啟用在打印國語稿時添加拼音注音，默認為關閉

### 打印或下載

- **粵語/國語模式**：點擊**打印**圖示（🖨️）來打印您的稿件
- **字幕模式**：點擊**下載**圖示（⬇️）來儲存為 `subtitle.txt` 檔案

## 使用提示

- 輸出內容會隨著您的輸入自動更新
- 如有需要，您可以直接編輯輸出區的文本，但輸出區的文本不會被保存，在輸入區有更改、輸出模式或設置有更改時，右側的輸出文本會被自動重新處理
- 字幕下載使用 UTF-16LE 編碼，以相容於視頻編輯軟體

## 系統需求

- 現代網頁瀏覽器（Chrome、Firefox、Safari 或 Edge）
- 已啟用 JavaScript
